Editor options

Is not necessary any lock button, when you edit a function this is automatic locked by your user.

1. Unlock

Open a popup with info about our locked functions. You can unlock your functions here.

2. Refresh

Load code editor with the last info. This operation is automatic while code is being edited.

3. Save

Save code to the historical. Code version is automatic-saved when you push enter, semicolon or backspace. You can save your version in this option.
